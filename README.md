# Personal-Portfolio-Website
This project is a responsive personal portfolio website developed using HTML and CSS. It includes Home, About, Skills, Projects and Contact sections with smooth navigation. The website showcases my profile, technical skills, projects and contact information through a clean and user friendly interface.This project was created as part of the Nexora UI Frontend Development Internship Week 1 task.
